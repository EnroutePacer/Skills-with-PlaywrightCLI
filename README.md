# Skills with PlaywrightCLI

Skills practical for automatizing Web workflow, cooperating with Playwright cli.  (Token Saving)  

The list is still growing...

## Pre

## Skills
* [academic](#Academic)
* [movie](#movie)

### Academic

```pwsh
/academic
```
Tell your agent a specific **academic field** you want to look into, to get A detailed reply including ——
- **current achievements** 
- **blank researching part** 
- **suggestions for new research**.

### Movie

```pwsh
/movie
```
面向国内观众的免费影视资源检索 skill，输入影视名字（中文、英文、别名），即可检索获得多个可播放的高质量链接。  

**Must** —— 
- **turn off proxy**
- **powershell**
